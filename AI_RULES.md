# AI Rules & Guidelines

1. **NO GREEN**: Never use the color green anywhere in the UI. If a library brings green, override it.
2. **Brand Palette**: Use `#FDF9F0` (warm cream) as the primary base. Complemented with gold or earth tones.
3. **Imagery**: ALL visual assets, people, and lifestyle imagery must reflect an authentic Indian context. No US-centric imagery.
4. **Architecture Constraint**: Use Next.js (App Router).
5. **Product Detail Fidelity**: Always show a minimum of 3-4 image angles for products.
6. **Feature Flags**: 
    - Partner With Us button must NOT be on the Consultation page.
    - Admin Panel must remain hidden and only accessible via tapping the footer logo.
7. **Autonomy**: Do not ask for user intervention when encountering trivial errors; attempt to fix them based on logs. Follow the workflow defined in `skills/development.md`.
